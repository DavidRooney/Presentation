Web Compiler

A tool used to compile SASS, LESS, JSX, ES6 and CoffeeScript.

Saving a source file triggers re-compiling automatically.

Successfully used on The Study Network, My Prize Draws, Lease Plan Go

Useful links:

  • Github
  • Visual Studio Gallery
D

Bundler & Minifier

A tool used to Bundle CSS, JavaScript or HTML files into a single output file or into a minified version.

Saving a source file triggers re-bundling automatically.

Option to convert to Gulp.

Successfully used on The Study Network to minifiy Javascript files.

Useful links:

  • Github
  • Visual Studio Gallery
D

GhostDoc

Allows you to quickly add comments to code. Remembering one shortcut saves you a lot of time!

Comments based on StyleCop rules!

Postman

  • Chrome app.
  • Create and send any HTTP request using the Postman request builder.
  • All requests quickly accessed in history and can be saved into collections.
  • Paid version - Testing and team sharing etc.
D
S

Catlight

  • Similar to TFS build notifications tool.
  • Works with both TFS and Visual Studio Online/Team Services.
  • Nicer user interface.
S

Google Keep

A simple note keeping app.

Lots of similar apps out there. e.g: Evernote & Microsoft OneNote

Benfits / Why I prefer:

  • Free
  • Desktop App, Mobile Apps (iOS, Android), Web version
  • Links to gmail account
  • Simple to use
  • Simple organize using labels, colours and archiving
  • Scheduled reminders
  • Can add images to individual notes

Useful links:

  • Website
  • Desktop App
D & E
D

Todoist

A checklist app.

Lots of similar apps out there. e.g: Google Keep, Microsoft OneNote, JIRA

Benfits / Why I prefer:

  • Free
  • Desktop App, Mobile App, Web version
  • Very simple to use
  • Simple organise using categories
  • Scheduled reminders
  • Collaboration - invite people to checklists. Assigning items to individuals
  • Group items into 1 item
  • Works on any device
  • Web version

Links to a gmail account so can set up personal and work versions very easily and switch between the 2.

Useful links:

  • Website
  • Desktop App

Instapaper

Benfits:

  • Can add notes to items saved including highlighting a single line.
  • Mobile App
  • Web version

  • Wappalyser

Web Developer

A chrome plugin that adds a lot of useful web development functions at the click of a button.

Some Functions:

  • Disabling javascript, Css
  • Quick Form Functions like clearing all check boxes, display passwords, auto populate form
  • Quickly clear cache, history, cookies
  • Quickly outline Headers, divs, any absolute positioned elements
  • Responsive window sizes

Useful links:

  • Chrome Extension
D

TestMyCss

Very quick online tool for analysing CSS. Simply pass it the css url and press GO.

Useful links:

D

Regex101

Very useful tool for testing regex, gives good explanation of your regex and matches. Quick reference panel in the bottom right particularly useful.

S

CodePen

A Front end developer playground - A free online tool that allows a user to create snippets of front end code.

Community based:

  • Very large community
  • Follow favourite users
  • "Like" Pens for viewing later
  • "Fork" pens to try yourself
*The inspiration for this presentation was taken from a CodePen that used Impress.JS and also found the header font from a CodePen.

Features:

  • Easily Add Html, Css and Javascript to create a snippet
  • Has compilers for HTML (e.g. Jade), CSS (e.g. SASS) and Javascript (e.g. CoffeeScript)
  • Very easy to add external libraries to the code. e.g. Bootstrap CSS, Jquery

Useful links:

  • Website
  • My CodePen
